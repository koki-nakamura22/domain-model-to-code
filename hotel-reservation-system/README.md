# ホテル予約システム ドメインモデル

ドメインモデル図からコードを生成するデモプロジェクト。

## ドキュメントのプレビュー

ブラウザ上でMarkdownファイル（Mermaid図含む）をプレビューするには、以下の2つが必要です。

1. **`md_server.py` の起動** — ローカルでMarkdownファイルを配信するサーバー
2. **ブラウザ拡張機能** — Markdownをレンダリングする拡張機能（例: [Markdown Viewer](https://chromewebstore.google.com/detail/markdown-viewer/ckkdlimhmcjmikdlpkmbgfkaikojcbjk)）

VSCodeを利用する場合は、Mermaidプレビュー対応の拡張機能（例: Markdown Preview Mermaid Support）でも閲覧可能です。
